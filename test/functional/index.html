Usage

In EXAMINE mode, use the left mouse button to rotate, the middle mouse button for pan and the right for zoom. If the web page has scroll bars and autoscrolling is enabled, pan currently does not work yet. As a workaround, you can disable autoscrolling by unchecking the Use autoscrolling checkbox in the browser options, as is shown in the screenshot below (for the Firefox case). Alternatively, use the Shift (rotate)/ Ctrl (Pan)/ Alt (Zoom) key in combination with a mouse button to switch between the three modes.
